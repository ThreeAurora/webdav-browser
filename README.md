# 📦 LazyCleaner | 躺平删图⚡极简文件管理🛋

个人自用，介绍随便写了。

一个极度简陋的 WebDAV 文件管理工具。

它不追求功能全面，只为让我能**舒舒服服躺在床上，用手机快速预览并删减电脑里的图片和视频**😊。

**感觉足够了，所以就没再继续开发。**

# ✨ 为什么做这个？

市面上的 WebDAV 工具（如 Alist）功能强大，但：

- 🚫 **不让无确认删除**（删个图弹窗问三遍，急死人）。
- 🎯 **不是专为“删减资源”设计**，操作路径长。

于是，这个“偏科”的工具诞生了。它**激进地砍掉了所有非必要功能**，只留下能让你最快完成“挑选-删除”的流程。

# 🎯 核心特点

## 1. 🛋 真正的“躺平”体验

- 手机直连电脑硬盘，窝在沙发或床上就能浏览、清理全盘资源。

## 2. ⚡ 秒删，无需确认

- 预览界面提供醒目的删除按钮（位置可调）。
- **点击即删除**，无二次确认弹窗打断心流。
- 删除的文件会自动移入当前硬盘的**独立回收站**，防手滑。

## 3. 🖐 直觉触控，刷得飞起

- **图片/视频**：点击左右区域或上下滑动，像刷短视频一样切换文件。
- **视频专属**：
	- 点**左侧**：快退5秒
	- 点**右侧**：快进5秒
	- 点**中间**：暂停/播放
- 手势少而精，学习成本为零。

## 4. 🗂 每盘一个“回收站”，安全又高效

- 删除的文件会移到对应磁盘根目录的 **`.webdav_trash`** 文件夹。
- 需要进入对应盘符（如`C盘`）才会在右上角显示**该盘的回收站入口**。
- 支持**还原**（自动处理重名）和**清空**。
- 同盘移动，极致速度。

## 5. 🎨 极简预览，零干扰

- **图片**：无缝浏览，切换流畅。
- **视频**：点开即播，**没有额外的控制层遮挡画面**，给你最干净的观看界面。
- 无用的成功提示、过渡动画？不存在的。

## 6. 📁 基础但够用的文件管理

- 丰富的文件类型图标（📕 PDF、📝 Word、🎵 音频等）。
- 支持排序（名称、类型、大小、日期，正反序）。
- 支持分页（20/50/100/200 每页）。
- 支持三种视图（▦网格/☰列表/▥瀑布流）
- 可设置显示隐藏文件（但回收站 `.webdav_trash`永不可见）。

# 📖 详细使用手册（小白友好版）

## 第一步：在电脑上架设文件服务器

1. **下载工具**：前往 Dufs 项目页，下载 `dufs.exe`。（https://github.com/sigoden/dufs）
2. **准备目录**：在电脑上新建一个文件夹，名字随意，将`dufs.exe`放入。
3. **创建“总入口”**：再新建一个子文件夹，名字随意，如 `AllDrives`。**这个文件夹将作为你手机访问所有硬盘的“总入口”**。
4. **创建磁盘链接**（关键步骤）：
	- 以**管理员身份**打开 **命令提示符 (CMD)**。
	- 一条条输入以下命令，为你的每个硬盘创建链接（请根据你的实际盘符修改）
		- mklink /J "D:\FileServer\AllDrives\C盘" "C:\"
		- mklink /J "D:\FileServer\AllDrives\D盘" "D:\"
		- mklink /J "D:\FileServer\AllDrives\E盘" "E:\"
	- 完成后，打开 `D:\FileServer\AllDrives`文件夹，应该能看到 `C盘`、`D盘`等快捷方式。
5. **创建启动脚本**：
	- 在 `D:\FileServer`文件夹内，新建一个文本文件，改名为 `start.bat`。
	- 右键用记事本编辑，粘贴以下内容并保存：
		- `dufs.exe AllDrives -A -b 0.0.0.0 -p 5000 --allow-symlink`
	- AllDrives就是你刚创建的“总入口”文件夹。
	- -p 5000是端口号，可自定义。
	- -A允许所有操作，--allow-symlink允许访问刚才创建的符号链接。

## 第二步：连接手机与电脑

1. **共享网络**：用电脑开启**移动热点**，让手机连接这个热点。*（目的是让两者处于同一局域网，不建议电脑连手机热点）*
2. **查找电脑IP**：
	- 在电脑上打开 CMD，输入 `ipconfig`并回车。
	- 找到 **“无线局域网适配器 WLAN”** 或类似名称下的 **IPv4 地址**，例如 `192.168.137.1`。**记下它**。
3. **启动服务器**：双击刚才创建的 `start.bat`文件。会弹出一个黑色窗口，**不要关闭它**。

## 第三步：在手机上使用本APP

1. **安装并打开**本APP。
2. **配置连接**：
	- 地址栏填写：`http://[你的电脑IP]:5000`（例如：`http://192.168.137.1:5000`）
	- 注意是 **`http`**，不是 `https`。
3. **开始使用**：连接成功后，你就能看到 `C盘`、`D盘`等目录，点进去即可浏览、预览、秒删文件。
4. **回收站在哪？**：需要先进入某个具体的盘符（如`C盘`），然后才能在界面**右上角**找到该盘的回收站入口。

## 🔁 日常使用流程（简化版）

1. 电脑上双击 `start.bat`。
2. 手机连上电脑的热点。
3. 打开APP（会自动填充上次的连接记录）。
4. 开刷，开删。
